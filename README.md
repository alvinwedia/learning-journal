# 📓 Jurnal Belajar: Reverse Engineering

Repository ini dibuat untuk mendokumentasikan perjalanan, proses, dan progres belajar saya selama satu semester dalam mata kuliah Reverse Engineering.

## 🎯 Tujuan Pembelajaran
* Membiasakan diri menulis dokumentasi teknis yang terstruktur.
* Mencatat kendala (error) yang ditemui selama praktikum dan bagaimana cara saya menyelesaikannya.
* Melacak perkembangan pemahaman saya dari minggu ke minggu.

## 📂 Struktur Repository
* `docs/` : Berisi catatan mingguan saya.
* `scripts/` : Berisi script sederhana yang saya buat untuk membantu otomatisasi belajar.
* `resources/` : Kumpulan tautan, *cheat sheet*, dan referensi yang saya baca setiap minggunya.
